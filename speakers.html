Workshops:

Speaker: Tom Steele (@_tomsteele)
Title: Intro to Node
Length: 2.5 hours
Requirements: Participants will preferably have a laptop with OSX or Debian based Linux distro such as Ubuntu (VM is fine). Others will probably work but at your own risk, it would be a bad idea to use Windows. Knowledge of how to use a text editor such as Vim and some prior experience with a scripting language such as ruby or python will be helpful, but you don't need to be an expert. If they would like to use an IDE, the best one for Node is JetBrains' WebStorm.
